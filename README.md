Angular Pilot Project
#1 환경설정
1. Angular Pilot 을 실행하기 위해서 다음이 필요함
  - node
  - grunt
  
2. grunt 설치 법
npm install -g grunt
npm install -g grunt-cli

3. 프로젝트 콘솔에서 다음 명령어 실행
npm install

4. 프로젝트 빌드
grunt

5. watch를 통한 자동 빌등
grunt watch

6. Node Express 실행
nodemon server
